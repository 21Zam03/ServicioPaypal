<h1 align="center"> Servicio de Pago con Paypal</h1>
<p>Este proyecto consiste en una aplicacion simple desarrollada en Java con spring boot utilizando la dependencia com.paypal.sdk para gestion de pagos con paypal. Esta aplicacion es capaz de realizar el proceso de pasarela de pago con paypal de manera eficiente y util.</p>

## Caracteristicas del proyecto
  - `Pasarela de pago`: Ofrece la capacidad de utilizar paypal como opcion de pago y utilizar su api para dicha funcion.
  - `com.paypal.sdk`: Configuracion de la dependencia com paypal sdk para la pasarela de pagos con paypal.

## Versiones y tecnologias importantes para ejecutar el programa
  - `Jdk`:  Jdk version 17 de preferencia
  - `IDEA`:  Un idea para poder ejecutar el proyecto de preferencia netbeans

## Instrucciones de uso
  1. [Descargar](#) o clonar este repositorio: git clone https://github.com/21Zam03/ServicioPaypal
  2. Abrir el proyecto en tu entorno de desarrollo Java preferido
  3. Create una cuenta de paypal para desarrolladores "https://developer.paypal.com/home/".
  4. Mira el siguiente video "" para la configuracion de la cuenta y credenciales en el archivo properties.
  5. Ejecuta en consola el comando "mvn spring-boot:run" para iniciar el proyecto y probar la pasarela de pago.
